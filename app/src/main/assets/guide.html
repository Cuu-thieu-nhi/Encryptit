1. Đăng nhập

Người dùng có thể đăng nhập bằng tài khoản Firebase (tạo đơn giản bằng chức năng Register) hoặc đăng nhập bằng tài khoản Google/Facebook (coming soon)


2. Menu

Có các tính năng về tài khoản và ứng dụng, các tính năng về tài khoản sẽ chỉ khả dụng với người dùng Firebase


3. Đổi mật khẩu


4. Đổi mật ảnh đại diện và tên hiển thị



5. Phản hồi vể ứng dụng


6. Thông tin về ứng dụng và tác giả


7. Thêm file để mã hóa

Ấn nút + để thêm file cần mã hóa. Ứng dụng có thể mã hóa gần như tất cả các file trên điện thoại Android. Ấn nút - để giải mã tất cả các file đã mã hóa





8. Click vào ảnh để xem chi tiết


9. Nhấn GIỮ ảnh để chọn


Nếu NHẤN vào ảnh đã ĐƯỢC CHỌN, sẽ hiện hộp thoại giải mã các ảnh đã chọn


10. NHẤN vào file để chọn


Nếu NHẤN GIỮ vào file đã ĐƯỢC CHỌN, sẽ hiện hộp thoại giải mã file ảnh đã chọn


11. Lọc danh sách file theo tùy chọn
